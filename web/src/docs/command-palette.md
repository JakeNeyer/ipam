# Command palette

The command palette provides quick search, navigation, and resource creation from anywhere in the app. Open it with `⌘K` (Mac) or `Ctrl+K` (Windows/Linux).

<img src="/images/command-palette-dark.png" alt="Command palette (dark mode)" class="screenshot-dark" />
</p>

- **Search** — Find environments, blocks, and allocations by name.
- **Navigate** — Jump to any page (`Dashboard`, `Environments`, `Networks`, etc.).
- **Create** — Start creating an environment, block, or allocation directly from the palette.
- **Keyboard** — Use `↑` `↓` to move selection and `Enter` to confirm. Press `Esc` to close.
